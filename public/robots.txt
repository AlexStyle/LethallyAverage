# REAL TALK (this is a security site, so let's be honest about what this file
# actually does): robots.txt is a REQUEST, not a wall. It's the honor system.
# Well-behaved crawlers read it and comply. A scraper running curl, or a bot
# that lies about its user-agent, ignores every line below and always will.
#
# Anything a browser can load, a person can copy. If you ever need to ACTUALLY
# block bots, you need something that can refuse a request in real time —
# a reverse proxy / CDN / WAF in front of the site (Cloudflare's free tier can
# do this). A static host can only ask nicely. Now you know the difference.
# ---------------------------------------------------------------------------
